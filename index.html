Kord-box

software-assisted harmonic exploration

Download this project as a .zip file Download this project as a tar.gz file

Get your theory on.

Harmonic theory is all fancy-like, but don't trip - here is some software to do the boring math for you as you explore chord voicings. Don't know what you're doing? It's all good - just press the buttons, and you can do no wrong.

W8 wot about minor harmony?

So complicated right? Not really - switch scales using the buttons atop each column. Drop into harmonic minor to slap folks with that V chord. Mix major and minor scales to pull on people's ear-holes.

Yes, this is very awesome.

Right? When you're feeling it, export to MIDI and load up into your trusty sequencer.
